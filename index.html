O aplikácii

RozvozPlan v1.0 · Lokálny dispatcher tool

Single-file HTML. Beží lokálne v prehliadači. Nevyžaduje server ani API key.

Mapy: Leaflet + OpenStreetMap (free)

Geocoding: Nominatim (free, 1 req/s)

Navigácia: Google Maps URL (bez API key)

QR: qrcodejs · PDF: jsPDF · Drag: SortableJS

GDPR: všetky dáta zostávajú lokálne v tvojom prehliadači.
